# Revendis Visual Identity (Official Freeze)

## Status
- Version: `1.0`
- Locked at: `2026-03-05`
- Scope: web app, mobile app, public storefront, sales assets.

## Brand Core
- Brand essence: `Confianca operacional para vender mais, com controle de ponta a ponta.`
- Symbol: `R monogram` with three motion planes and technical edges.
- Personality: `direct`, `safe`, `commercial`, `operational`.

## Color Direction
- Primary base: deep night blue.
- Action accents: electric blue and cyan.
- Monetary positive accents: mint/green.
- Warning/error: warm orange and red.

## Rules
1. Do not create alternate palettes outside the official tokens.
2. Keep green for money/approval/commercial growth contexts.
3. Keep blue/cyan for action, navigation and data highlights.
4. Do not use random gradient combinations in core screens.
5. New modules must inherit the same token scale.

## Product Signature
- `Revendis Market`: green accent (`growth/commercial`).
- `Revendis Analytics`: blue accent (`insight/reading`).
- `Revendis Pay`: mint accent (`payment/settlement`).

## Governance
- Single source of truth in code: `apps/web/app/lib/revendis-design-system.ts`.
- Any visual change requires:
  - update in token source,
  - update in docs,
  - explicit approval in product/design review.
